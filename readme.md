# 🧠 Duplicate Question Pair Detector

A simple NLP-based web app that predicts whether two questions are **Duplicate** or **Not Duplicate** using Machine Learning.

---

## 🚀 Features
- Detects semantic similarity between questions  
- Built with Streamlit UI  
- Uses NLP + Feature Engineering  
- Fast and interactive  

---

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Scikit-learn  
- FuzzyWuzzy  
- BeautifulSoup  

---

## 📂 Project Structure
```

streamlit-app/
├── app.py
├── helper.py
├── model.pkl
├── cv.pkl
├── requirements.txt
├── setup.sh
├── Procfile

````

---

## ⚙️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
````

---

## 🧪 Example

**Q1:** What is machine learning?
**Q2:** What is meant by machine learning?

➡️ Output: Duplicate
